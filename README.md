## Descripción
Este proyecto es una página web básica desarrollada como parte del curso Front-End de <strong>TalentoTech</strong>. La página está estructurada con HTML semántico y utiliza las etiquetas &apos;&lt;header&gt;&apos;, &apos;&lt;main&gt;&apos; y &apos;&lt;footer&gt;&apos; para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript.

## Features
- Formulario de contacto con Formspree: se creó un proyecto de Formspree para recibir formularios de contacto implementados en el proyecto. La dirección web del proyecto se carga como argumento 'action' del formulario. Formspree procesa automáticamente las entradas y permite revisarlas o borrarlas según se necesite.
